# Tuning Tool Shop

TYPO3 13 Shop Extension für Chiptuning Geräte und Zubehör.

## Features

- Produktkatalog mit Kategorien und Herstellern
- Produktdetailseiten mit Bildern, Videos und Dokumenten
- Warenkorb-Funktionalität
- One-Page-Checkout
- Verschiedene Zahlungsmethoden
- Bestellverwaltung

## Installation

```bash
composer require hamstahstudio/tuning-tool-shop
```

## Plugins

- **ProductList**: Zeigt eine Liste von Produkten
- **ProductShow**: Zeigt Produktdetails
- **Cart**: Warenkorb-Ansicht und -Verwaltung
- **Checkout**: One-Page-Checkout-Prozess

## Anforderungen

- PHP >= 8.2
- TYPO3 >= 13.4
