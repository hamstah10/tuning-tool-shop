================================================================================
PRODUCT API - ENDPOINTS
================================================================================

GET /api/product
  → Alle Produkte laden

GET /api/product/{uid}
  → Einzelnes Produkt nach UID

GET /api/product/active
  → Nur aktive Produkte

GET /api/product/search?term=...
  → Nach Suchbegriff suchen

GET /api/product/recent?limit=10
  → Neueste Produkte (default limit: 10)

GET /api/product/sku?sku=...
  → Produkt nach SKU-Nummer suchen

================================================================================
ORDER API - ENDPOINTS
================================================================================

GET /api/order
  → Alle Bestellungen laden

GET /api/order/{uid}
  → Einzelne Bestellung nach UID

GET /api/order/number?number=...
  → Bestellung nach Bestellnummer

GET /api/order/email?email=...
  → Bestellungen nach Kundenemails

GET /api/order/status?status=...
  → Bestellungen nach Status filtern
  Status: 0=Neu, 1=Bearbeitung, 2=Bestätigt, 3=Versendet, 4=Abgeschlossen, 5=Storniert

GET /api/order/recent?limit=10
  → Neueste Bestellungen (default limit: 10)

GET /api/order/user?user=...
  → Bestellungen eines Frontend-Benutzers

VERWENDETE TECHNOLOGIEN

Framework:          TYPO3 13.4
PHP Version:        8.2+
REST API Extension: nnrestapi
ORM:                Extbase/Doctrine
Response Format:    JSON

================================================================================
INSTALLATION & AKTIVIERUNG
================================================================================

1. nnrestapi Extension installieren (bereits vorhanden):
   composer require nng/nnrestapi

2. API-Klassen sind automatisch registriert

3. TYPO3 Cache leeren:
   php vendor/bin/typo3 cache:flush

4. Endpoints sind erreichbar unter:
   https://example.com/api/product
   https://example.com/api/order

================================================================================

================================================================================

Erfolg:
