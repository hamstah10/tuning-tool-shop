Frontend Features
=================

Shop Page
---------

The shop page displays all available products with:

- Product images
- Product titles and descriptions
- Pricing information
- Stock availability
- Product rating and reviews
- Quick add-to-cart buttons

Browsing Products
~~~~~~~~~~~~~~~~~

Users can browse products by:

- **Categories** - Filter by product category
- **Manufacturers** - Filter by brand/manufacturer
- **Tags** - Browse by tags
- **Search** - Full-text search across product titles and descriptions

Product Detail Page
-------------------

Clicking on a product shows:

- Full product description
- Multiple product images (if available)
- Detailed specifications
- Customer reviews and ratings
- Related products
- Availability and shipping information
- Add to cart button with quantity selector

Shopping Cart
-------------

The shopping cart allows customers to:

- View all items
- Update quantities
- Remove items
- Apply discount codes
- View cart total with taxes and shipping
- Proceed to checkout

Checkout Process
----------------

The checkout process includes:

1. **Shipping Address** - Enter or select shipping address
2. **Billing Address** - Option to use shipping address
3. **Shipping Method** - Select from available options
4. **Payment Method** - Choose payment provider
5. **Order Review** - Final confirmation
6. **Payment** - Redirect to payment provider

Payment Methods
~~~~~~~~~~~~~~~

Available payment methods:

- **PayPal** - Secure payment via PayPal
- **Stripe** - Credit/debit card payments
- **Klarna** - Buy Now Pay Later option

Order Management
----------------

Authenticated customers can:

- View order history
- Track order status
- View order details
- Download invoices (if available)
- Return orders (if applicable)

Account Section
---------------

Users can manage their account:

- View profile information
- Manage saved addresses
- View order history
- Manage payment methods
- Download invoices
