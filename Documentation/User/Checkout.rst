========
Checkout
========

The checkout process guides you through completing your purchase.

Checkout Steps
==============

Step 1: Shipping Address
------------------------

Enter your shipping address:

- **First Name** - Required
- **Last Name** - Required
- **Company** - Optional
- **Street Address** - Required
- **City** - Required
- **State/Province** - If applicable
- **Postal Code** - Required
- **Country** - Required
- **Phone Number** - Required
- **Email Address** - Required

Address Validation
~~~~~~~~~~~~~~~~~~

- Address is validated for completeness
- Error messages for invalid entries
- Suggestions for address correction

Saved Addresses
~~~~~~~~~~~~~~~

If logged in, you can:

- Select from saved addresses
- Add new address
- Edit address information
- Set default address

Step 2: Billing Address
-----------------------

Billing Address Options
~~~~~~~~~~~~~~~~~~~~~~~

- **Same as Shipping** - Use shipping address
- **Different Address** - Enter separate billing address
- **Use Saved Address** - Select from your saved addresses

Entering Billing Address
~~~~~~~~~~~~~~~~~~~~~~~~

If different from shipping:

- Enter all required fields
- Same validation as shipping address
- Can save for future use

Step 3: Shipping Method
-----------------------

Available Methods
~~~~~~~~~~~~~~~~~

Select your preferred shipping:

- **Method Name** - (e.g., DHL, FedEx)
- **Cost** - Shipping price
- **Estimated Delivery** - Expected arrival date
- **Description** - Service details

Shipping Cost
~~~~~~~~~~~~~

Displayed for each method:

- Base shipping cost
- Weight-based charges (if applicable)
- Zone-based pricing
- Free shipping threshold (if applicable)

Estimated Delivery
~~~~~~~~~~~~~~~~~~

Each method shows:

- Estimated delivery time
- Business days
- Delivery date range
- Cutoff time for same-day/next-day

Step 4: Payment Method
----------------------

Payment Options
~~~~~~~~~~~~~~~

Available payment methods:

- **PayPal** - Secure payment via PayPal
- **Stripe** - Credit/debit card
- **Klarna** - Buy Now Pay Later
- **Bank Transfer** - Direct payment (if enabled)

PayPal Checkout
~~~~~~~~~~~~~~~

1. Click PayPal button
2. Redirect to PayPal
3. Login to PayPal account
4. Review and approve payment
5. Return to shop

Stripe Payment
~~~~~~~~~~~~~~

1. Enter card details
2. Card is tokenized for security
3. Review order
4. Process payment
5. Confirmation page

Klarna Payment
~~~~~~~~~~~~~~

1. Select Klarna option
2. Review Klarna terms
3. Provide required information
4. Receive payment plan details
5. Complete payment

Step 5: Order Review
--------------------

Review Order Summary
~~~~~~~~~~~~~~~~~~~~

- Items ordered
- Quantity per item
- Price per item
- Line totals
- Subtotal
- Shipping cost
- Tax amount
- Order total
- Currency

Shipping Details
~~~~~~~~~~~~~~~~

Verify:

- Shipping address
- Shipping method
- Estimated delivery
- Contact information

Payment Details
~~~~~~~~~~~~~~~

Confirm:

- Payment method selected
- Amount to be charged
- Billing address

Making Changes
~~~~~~~~~~~~~~

If you need to change:

- Items: Go back to cart
- Shipping address: Edit address
- Shipping method: Select different option
- Payment: Choose different method

Completing Purchase
====================

Place Order
-----------

1. Review all information
2. Check terms and conditions
3. Click "Place Order" or "Complete Purchase"
4. Payment is processed
5. Order is created

Order Confirmation
~~~~~~~~~~~~~~~~~~

After successful order:

- Confirmation number displayed
- Order summary shown
- Confirmation email sent
- Account order history updated

Payment Processing
-------------------

Payment Status
~~~~~~~~~~~~~~

Orders can have status:

- **Pending** - Awaiting payment
- **Processing** - Payment received
- **Completed** - Order ready to ship
- **Cancelled** - Order cancelled

Receipt
-------

You receive:

- Email confirmation
- Order number
- Invoice (if available)
- Tracking number (when shipped)
- Return instructions

Troubleshooting
===============

Payment Failed
--------------

If payment fails:

1. Check card details
2. Verify sufficient funds
3. Try alternative payment method
4. Contact support if issue persists

Order Not Received
~~~~~~~~~~~~~~~~~~~

If order confirmation doesn't arrive:

1. Check email spam folder
2. Verify email address in account
3. Check order history in account
4. Contact customer support

Issues During Checkout
----------------------

If you encounter errors:

- Verify all required fields filled
- Check internet connection
- Clear browser cache
- Try different browser
- Contact support with error message

Order Tracking
==============

After Order
-----------

You can track your order:

- Login to account
- View order history
- Check order status
- View tracking number
- Get delivery updates

Email Notifications
~~~~~~~~~~~~~~~~~~~

Receive emails for:

- Order confirmation
- Payment confirmation
- Order shipped
- Out for delivery
- Delivery confirmation

Returns and Exchanges
======================

Return Policy
-------------

See terms for:

- Return eligibility
- Return window timeframe
- Return shipping costs
- Return process steps

Initiating Return
~~~~~~~~~~~~~~~~~

1. Login to account
2. Select order to return
3. Click "Return" or "Request Return"
4. Select items to return
5. Provide reason (if required)
6. Generate return label
7. Ship items back
8. Await refund processing

Refund Status
~~~~~~~~~~~~~

After return:

- Track return shipment
- View refund status
- Receive refund notification
- Check refund in account
