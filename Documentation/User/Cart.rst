==============
Shopping Cart
==============

The shopping cart allows you to manage items before checkout.

Adding Products
===============

Quick Add
---------

Click "Add to Cart" on any product to:

- Add product to your cart
- Select quantity
- Close the modal or continue shopping

Product Detail Add
~~~~~~~~~~~~~~~~~~

On the product detail page:

- Select product variant/options (if available)
- Set desired quantity
- Click "Add to Cart"
- Confirmation message appears
- Option to continue shopping or view cart

Cart Contents
=============

Viewing Your Cart
-----------------

Access your cart by:

- Clicking cart icon in header
- "View Cart" link after adding item
- Cart page in navigation

Cart Summary
~~~~~~~~~~~~

The cart shows:

- List of items
- Product images
- Product names
- Product variants (if applicable)
- Unit price
- Quantity
- Line total
- Remove button for each item

Cart Totals
-----------

Below the items:

- **Subtotal** - Sum of all items
- **Discount** - Applied coupon/discount
- **Shipping** - Estimated shipping cost
- **Tax** - Calculated tax
- **Total** - Final order amount

Modifying Cart
===============

Changing Quantities
-------------------

Update item quantity by:

- Typing new quantity in input field
- Using +/- buttons
- Click "Update Cart" to apply changes

Removing Items
--------------

Remove items by:

- Clicking "Remove" button next to item
- Confirmation appears
- Cart updates automatically

Applying Discount Codes
=======================

Adding Coupon
--------------

If you have a discount code:

1. Look for "Coupon Code" or "Promo Code" input
2. Enter the code
3. Click "Apply" or "Redeem"
4. Discount applies to total
5. Remove code if needed

Continuing Shopping
====================

Options
-------

- **Continue Shopping** - Return to shop to browse more
- **Update Cart** - Apply quantity changes
- **Proceed to Checkout** - Go to checkout process

Checkout Button
~~~~~~~~~~~~~~~

Proceed to checkout by:

- Clicking "Checkout" button
- Review shipping/billing address entry
- Select payment method

Empty Cart
==========

Clear Cart
----------

To clear entire cart:

1. Go to cart page
2. Look for "Clear Cart" or "Remove All Items" button
3. Confirm action
4. Cart is emptied

Continue Shopping
~~~~~~~~~~~~~~~~~

After clearing:

- Browse products again
- View categories
- Use search function

Cart Persistence
================

Cart Storage
------------

Your cart is:

- Stored in your browser (for guests)
- Saved to account (for logged-in users)
- Persistent across sessions
- Synchronized on login

Abandoned Cart
--------------

If you leave without checking out:

- Cart items are saved
- Automatically restore on return
- Reminder emails (if configured)
- Items held for limited time

Cart Features
=============

Wishlist Integration
--------------------

From cart you can:

- Move items to wishlist
- Save for later purchase
- Access wishlist anytime

Save for Later
--------------

- Move items to save for later
- Items remain in account
- Add back to cart when ready

Share Cart
----------

- Generate shareable link
- Share with others
- Track shared carts
