======
Orders
======

View and manage your orders.

Order History
=============

Accessing Order History
-----------------------

Login to your account to view:

- All your orders
- Order date
- Order number
- Order status
- Order total
- Actions (view, track, return)

Order List
~~~~~~~~~~

View all orders with:

- Most recent first
- Sorting options
- Filtering options
- Search functionality

Order Details
=============

Viewing Order Details
---------------------

Click on an order to see:

- Order number
- Order date
- Order status
- Items ordered with quantities and prices
- Subtotal, shipping, tax, and total
- Delivery address
- Estimated delivery date

Order Status
------------

Possible order statuses:

- **New** - Order received
- **Processing** - Preparing for shipment
- **Shipped** - In transit to you
- **Delivered** - Successfully delivered
- **Cancelled** - Order was cancelled

Order Items
~~~~~~~~~~~

For each item in order:

- Product name
- Product image (if available)
- Product SKU
- Quantity ordered
- Unit price
- Line total
- Option to view product

Payment Information
~~~~~~~~~~~~~~~~~~~

View payment details:

- Payment method used
- Payment date
- Payment status
- Transaction ID (if available)
- Payment confirmation

Shipping Information
--------------------

Shipping Details
~~~~~~~~~~~~~~~~

- Shipping address
- Shipping method used
- Estimated delivery date
- Actual delivery date (if delivered)

Tracking
~~~~~~~~

If order is shipped:

- Tracking number
- Carrier information
- Tracking link
- Estimated delivery
- Current location (if available)

Returns
-------

Return Status
~~~~~~~~~~~~~

If return initiated:

- Return status
- Return date
- Items being returned
- Return shipping label
- Refund status

Initiating Return
~~~~~~~~~~~~~~~~~

On order detail page:

1. Click "Request Return"
2. Select items to return
3. Provide return reason
4. Receive return label
5. Ship items back
6. Track return

Invoices and Documents
======================

Downloading Invoice
-------------------

If available, download:

- Invoice PDF
- Packing slip
- Shipping label
- Return label

Document History
~~~~~~~~~~~~~~~~

Access all documents:

- Download anytime
- Print documents
- Share with accountant
- Keep for records

Order Actions
=============

Available Actions
-----------------

Depending on order status:

- **Track Order** - View tracking information
- **Request Return** - Initiate return/exchange
- **Cancel Order** - Cancel if eligible
- **Contact Support** - Get help with order
- **Print Order** - Print order details
- **Download Invoice** - Get invoice PDF

Filtering and Sorting
=====================

Filter Options
--------------

Filter orders by:

- Status (New, Processing, Shipped, etc.)
- Date range
- Payment status
- Amount range

Sorting
~~~~~~~

Sort orders by:

- Most recent first
- Oldest first
- Highest price
- Lowest price
- Order number

Search
~~~~~~

Search orders by:

- Order number
- Date
- Amount
- Item name

Notifications
=============

Order Notifications
-------------------

You receive email notifications for:

- Order confirmation
- Payment confirmation
- Order processing
- Order shipped
- Out for delivery
- Delivery confirmation

Notification Settings
~~~~~~~~~~~~~~~~~~~~~

Manage notifications in account:

- Enable/disable emails
- Change email address
- Frequency preferences
- Notification types

Reordering
==========

Quick Reorder
-------------

Reorder from previous order:

- Click "Reorder" button
- Items added to cart
- Proceed to checkout
- Quantity may vary if out of stock

Requesting Similar Items
~~~~~~~~~~~~~~~~~~~~~~~~~

Create new order with:

- Similar products
- Different quantities
- Different variants
- Different combinations

Account Integration
===================

Connected to Account
--------------------

Order history shows:

- All purchases from account
- All shipping addresses used
- All payment methods used
- Account profile information

Multiple Orders
~~~~~~~~~~~~~~~

If you have multiple accounts:

- Each account separate history
- Login to view specific orders
- Cannot see others' orders

Privacy
-------

Your order information:

- Private to your account
- Protected by password
- Secure HTTPS connection
- Not shared without consent
