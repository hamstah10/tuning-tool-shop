====
Shop
====

The Shop module is the main interface for customers to browse and discover products.

Browsing Products
=================

Main Shop Page
--------------

The main shop page displays:

- **Featured Products** - Highlighted products
- **New Products** - Recently added items
- **Category Menu** - Navigate by category
- **Manufacturer List** - Filter by brand
- **Search Box** - Search for products

Product Listings
~~~~~~~~~~~~~~~~

Products are displayed in a grid or list format with:

- Product image
- Product name
- Price
- Stock status
- Quick view button
- Add to cart button

Filtering and Sorting
---------------------

Users can filter products by:

- **Category** - Select from product categories
- **Manufacturer** - Filter by brand
- **Price Range** - Set min/max price filter
- **Availability** - In stock / out of stock

Sorting options:

- **Newest** - Recently added products
- **Popularity** - Most viewed products
- **Price Low to High** - Ascending price
- **Price High to Low** - Descending price
- **Rating** - Highest rated products

Searching Products
------------------

Full-text search is available through:

- Main search box on shop page
- Dedicated search page
- Search by product name
- Search by description
- Search by SKU/product code

Search results include:

- Matching products
- Number of results
- Sorting options
- Filtering options

Product Detail View
===================

Clicking a product opens the detail page showing:

Images
------

- Main product image
- Thumbnail gallery
- Image zoom functionality
- Alternative product views

Product Information
-------------------

- Product name
- Product description
- Full specifications
- Product code/SKU
- Manufacturer information

Pricing
-------

- Regular price
- Special/sale price (if applicable)
- Pricing details
- Currency display

Stock Status
~~~~~~~~~~~~

- Stock availability indicator
- Available quantity (if configured)
- Shipping timeframe
- Backorder information

Customer Reviews
----------------

- Star rating
- Customer reviews
- Review form for verified buyers
- Average rating display

Related Products
~~~~~~~~~~~~~~~~

The page displays related products:

- Same category
- Same manufacturer
- Popular together
- Customer bought also bought

Actions
-------

- Add to cart button
- Add to wishlist button
- Share product button
- Print product button

Variants and Options
~~~~~~~~~~~~~~~~~~~~

If the product has variants:

- Select variant from dropdown
- View variant-specific pricing
- View variant-specific images
- View variant stock status

Categories
==========

Category Pages
--------------

Each category page shows:

- Category name and description
- Subcategories (if any)
- Products in category
- Filtering and sorting options

Manufacturer Pages
==================

Manufacturer Overview
---------------------

Browse all products from a specific manufacturer:

- Manufacturer information
- Manufacturer logo
- All products from manufacturer
- Sorting and filtering

Tags
====

Tag-based Navigation
--------------------

Browse products by tags:

- View products with specific tag
- View related tags
- Tag cloud view
- Popular tags

Popular Sections
================

Featured Products
-----------------

Highlighted premium or popular products displayed on:

- Shop homepage
- Category pages
- Sidebar widget

New Products
~~~~~~~~~~~~

Recently added products:

- Latest additions
- Newest first sorting
- Time-limited display
- Limited quantity

Special Offers
~~~~~~~~~~~~~~

Products with special pricing:

- Sale/promotion products
- Discount percentage display
- Limited time offers
