=======
Account
=======

Manage your account and preferences.

Account Profile
===============

Profile Information
-------------------

Update your profile:

- **First Name** - Required
- **Last Name** - Required
- **Email Address** - Used for login and notifications
- **Phone Number** - Contact number
- **Company Name** - Optional business name
- **Language** - Preferred language

Changing Password
~~~~~~~~~~~~~~~~~

To change your password:

1. Go to Account > Security
2. Enter current password
3. Enter new password (twice)
4. Click "Update Password"
5. Confirmation email sent

Password Requirements:

- Minimum 8 characters
- Mix of uppercase and lowercase
- Include numbers
- Include special characters

Addresses
=========

Saved Addresses
---------------

Manage your addresses:

- View all saved addresses
- Add new address
- Edit existing address
- Delete address
- Set default address

Adding Address
~~~~~~~~~~~~~~

Click "Add Address":

1. Select address type (Shipping/Billing)
2. Fill in address details
3. Click "Save"
4. Address available for checkout

Editing Address
~~~~~~~~~~~~~~~

Click "Edit" on address:

1. Modify fields
2. Click "Save Changes"
3. Updates reflected immediately

Default Address
~~~~~~~~~~~~~~~

Set default shipping/billing:

- Used automatically in checkout
- Change anytime
- Select different for each order

Payment Methods
===============

Saved Payment Methods
---------------------

Manage payment options:

- View saved cards
- Add new payment method
- Remove payment method
- Set default method

Adding Credit Card
~~~~~~~~~~~~~~~~~~

Click "Add Payment Method":

1. Enter card details
2. Card is tokenized (not stored in plain text)
3. Set as default (optional)
4. Confirm addition
5. Available for future purchases

Removing Payment Method
~~~~~~~~~~~~~~~~~~~~~~~

Click "Remove" on payment:

1. Confirm removal
2. Card is deleted
3. Cannot be recovered

Security
--------

Payment security:

- PCI-DSS compliant
- Encrypted storage
- No plain text storage
- Secure payment processing
