Dashboard
=========

Admin Dashboard Overview
------------------------

The admin dashboard provides:

- Order statistics
- Product inventory status
- Recent orders
- Customer information
- Quick actions

Dashboard Widgets
~~~~~~~~~~~~~~~~~

- **Orders Widget** - Recent orders and status
- **Products Widget** - Product inventory
- **Inventory Widget** - Stock levels
- **Revenue Widget** - Sales statistics

Manage your catalog:

- Create products
- Edit product details
- Manage variants
- Set pricing
- Control stock

Category Management
~~~~~~~~~~~~~~~~~~~

Organize products:

- Create categories
- Manage hierarchy
- Set category images
- Configure descriptions

Manufacturer Management
~~~~~~~~~~~~~~~~~~~~~~~

Manage brands:

- Create manufacturers
- Add logos
- Edit information

Customer Management
-------------------

View customer data:

- Order history per customer
- Contact information
- Preferences

Order Management
~~~~~~~~~~~~~~~~

Process orders:

- View all orders
- Update order status
- Process refunds
- Manage returns
- Send notifications

Payment Configuration

Setup payment methods:

Shipping Configuration
